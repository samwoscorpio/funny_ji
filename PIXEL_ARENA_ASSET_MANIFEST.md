# 潮汐遗迹：像素竞技场资产包 v2

## 用途与边界

这是供审核的**可拼接素材方向包**。它不修改现有战斗代码，也不是一张要直接铺到游戏里的固定背景。后续实现应继续读取现有 8×8 六边形地图数据，并按本清单将独立地形、功能物和提示层组合绘制。

## 固定层级

1. 深渊、水面、远景遗迹。
2. 外围悬崖、墙体和水流。
3. `hex-floor` 地面格。
4. 高草、巨石、圆形祭坛、能量水晶、火焰。
5. `thin-wall-edge`。
6. 移动、攻击、选中与路径覆盖层。
7. 队伍脚环、单位像素精灵与战斗效果。

## 与现有机制的对应

- **草丛**：使用 `terrain-tall-grass`，整格必须被高草覆盖，不能留出中央灌木加地砖的空白构图。相邻草丛格共享边上的草叶要彼此咬合，形成连续草地区域；孤立草丛同样应自然铺满自身六边形。是否隐藏敌方单位仍由现有可见性逻辑决定。
- **巨石**：视觉上占一个六边格；是否阻挡移动和攻击仍由地图的 `walls` 数据决定。
- **薄墙**：必须作为两格之间的边缘资产绘制，对应 `thinWalls`；不能作为单格障碍物。
- **能量水晶**：只标记 `energyTiles` 的当前位置，不包含任何随机逻辑。
- **据点**：使用低矮的圆形祭坛，只标记 `objective`；其连续占据、XP 支付和分数规则保持在现有逻辑中。
- **火焰**：只显示临时危险地表；伤害规则不属于该素材。

## 已确认的美术决定

1. 主色关系保持“深灰遗迹石材 + 青色能量水晶 + 金橙目标”。
2. 据点采用**地面圆形祭坛**：应是醒目的暖金橙视觉焦点，但不得呈现为方尖碑、高塔或建筑。
3. 草丛采用**整格高草**：每个草丛格填满六边形，连续草丛格需在共享边自然接合成片。
4. 悬崖水流、巨石、低矮薄墙、火焰与行动高亮延续 v1 的既定风格。
